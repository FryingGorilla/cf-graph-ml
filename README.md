# README

## Overview
This repository contains machine learning models and tools for analyzing CodeForces competitive programming data, including an Item Response Theory model and a graph neural network recommender system.

## Getting Started

To start the application, execute:
```bash
./run.sh
```

## Prerequisites
- Docker
- Docker Compose (optional, depending on setup)

## Dataset Information

**Important Note:** The IRT model was trained on lost data. For the latest and most reliable dataset, please refer to the updated dataset available here:

[CodeForces-Interactions Dataset on Hugging Face](https://huggingface.co/datasets/najrum/cf-interactions)

## Usage
Ensure Docker is installed and running before executing the startup script.
